# 我的口播口吻档案（模板）

> 把这个文件复制为 `my-voice-profile.md` 后**自己改写**。本模板是结构示例，里面的描述都是占位文字，必须替换成你自己账号的真实情况。
>
> 来源建议：抖音创作者后台 24 条以上历史视频摘要（用 `抓后台.bat` 拉到 `my-history-backend.md`）+ 你能下载到的视频转写片段。

## 账号定位

> 一句话说清楚：你的账号在帮谁解决什么问题。
> 模板示例：「我帮 [目标人群] 把 [具体麻烦] 拆解成 [可立刻执行的小步骤]」。
> 不是讲宏大趋势，也不是工具教程，而是把一件 [离钱/离结果最近] 的事拆开。

## 你的有效口吻

按你自己的真实风格填，常见维度有：

- 句式：直接 / 二人称 / 长句 / 短句 / 反问句多 / 金句多
- 高频本质句：写下你自己常用的口头禅（例：「本质就是」「说白了」「你要先搞清楚」）
- 用词风格：粗粝接地气 / 知识精英 / 邻家姐姐 / 商业咨询
- 案例偏好：你常拿什么场景举例（外卖店 / 家教中介 / 写字楼办公室 / 海外工作）
- 结尾习惯：行动命令 / 留悬念 / 反问 / 金句收

## 高播样本规律

填 3-5 条你账号已经跑出过的爆款，从里面提炼共性。每条记：

- 为什么观众愿意转发 / 收藏
- 用了什么结构（断言 → 案例 / 反共识 → 拆解 / 反问 → 引导）

### 2. ...

### 3. ...

## 低播样本问题

填 2-3 条你账号已经掉过的视频，分析为什么没起来。常见原因：

- 太泛 / 缺具体人群
- 只讲观点没讲怎么收钱
- 只讲恐惧没给出路
- 工具名堆砌
- 开头铺垫太久

## 写稿模板

至少给 2-3 个你自己常用的开稿框架。常见两种：

### 模板 A：强断言洗脑型

开头：[一句反共识断言，没有铺垫]
中段：[本质句 + 推理]
案例：[1 个具体场景]
结尾：[行动命令 / 反问 / 金句]

### 模板 B：赚钱思路拆旧型

开头：[攻击旧认知]
中段：[新框架]
案例：[小钱小闭环]
结尾：[先赚第一个 N 块]

## 写稿禁区

- 列你账号绝对不能碰的话术（避免被限流、避免与人设冲突）
- 例：不出现具体工具名 / 不喊百万 / 不替对标账号人设说话 / 不写成精英商业课

## 后续需要补充的素材（非数据类）

- 你能交付的产品 / 服务边界
- 哪些真实经历可以公开讲
- 这一期想打的具体人群
