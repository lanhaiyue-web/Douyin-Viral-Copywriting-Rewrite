# 快速开始

跟着这 6 步走，30 分钟从 clone 到第一次抓出后台数据。

这个仓库不带任何现成账号数据。你需要扫码登录自己的抖音创作者中心，自己填写口吻档案和对标账号；这些运行时文件都会被 `.gitignore` 排除，不会进入 GitHub。

## Step 1：装 Python 3.10+

[python.org/downloads](https://www.python.org/downloads/) 下载 3.10 或更新版本。

Windows 安装时**勾选 "Add python.exe to PATH"**。

## Step 2：装项目依赖

打开 PowerShell，cd 到本项目目录后跑：

```powershell
python -m venv .venv
.venv\Scripts\python.exe -m pip install --upgrade pip
.venv\Scripts\python.exe -m pip install -r requirements.txt
.venv\Scripts\python.exe -m playwright install chromium
```

> `playwright install chromium` 会下载约 150MB 的浏览器内核，用于抓抖音后台。等 1-2 分钟。

## Step 3：填 DeepSeek API Key

DeepSeek 用来给文案打分。

1. 到 https://platform.deepseek.com/ 注册并充值（最少 1 元，能用很久）
2. 创建 API Key
3. 新建文件 `%APPDATA%\baokuan-rewrite\.env`（PowerShell 里跑 `mkdir $env:APPDATA\baokuan-rewrite -Force` 建目录），填入：

   ```ini
   DEEPSEEK_API_KEY=sk-你的-key
   ```

> 不要把 key 填到项目目录的 `.env`。项目目录的 `.env` 已 .gitignore 但**习惯上**真实 key 一律放 `%APPDATA%`，永远不会进 git。

## Step 4：扫码登录抖音创作者中心

双击项目根目录的 `扫码登录抖音.bat`。

会弹出一个 Chromium 浏览器窗口，**用手机抖音 App 扫窗口里的二维码**。扫完后 cookies 自动写到 `.auth/`（已 .gitignore），下次刷新数据不用再扫。

> session 大约 3-5 天没用会过期，过期了再跑一次这个 bat。

> 已知：扫完后程序可能提示 `[登录] 超时未检测到登录态`——是误报，cookies 已写盘，直接进下一步。

## Step 5：抓一次后台数据

双击 `抓后台.bat`。

会再弹 Chromium 窗口（这次复用登录态自动跑），约 30 秒后看到：

```
[写入] context/my-history-backend.md（N 条）
[最近 5 条]
  - <aweme_id> | 2026-XX-XX HH:MM | XX.Xs | 播放 NNN | <标题>
  ...
[完成]
```

数据已经写入 `context/my-history-backend.md`，可以打开看看。

如果要顺便抓最近 2 条视频的评论：

```bat
抓后台.bat --with-comments
```

评论会落到 `context/comments/<aweme_id>.md`。

## Step 6：填好你的口吻档案

打开 `context/my-voice-profile.template.md`，**另存为 `my-voice-profile.md`**（不要 template 后缀），按里面的结构填：

- 账号定位
- 高播 / 低播规律
- 写稿模板
- 写稿禁区

填这一份大概需要 1-2 小时，但是是文案质量的最大杠杆——AI 写出的稿子 80% 来自这份档案。

同样，对标账号至少选 3 个，每个抓 30 条数据填进 `context/benchmarks/<对标账号>.md`。

## Step 7：开始写文案

```bat
start_app.bat
```

浏览器自动打开 http://localhost:8501，三个 Tab：评分 / 改写 / 原创。

## 常用命令速查

| 操作 | 命令 |
|---|---|
| 启动写文案网页 | `start_app.bat` |
| 抓最新后台数据 | `抓后台.bat` |
| 抓后台 + 最近 2 条评论 | `抓后台.bat --with-comments` |
| 抓后台 + 最近 5 条评论 | `抓后台.bat --with-comments --top 5` |
| 抓更多历史视频 | `抓后台.bat --limit 50` |
| 抖音登录过期重登 | `扫码登录抖音.bat` |
| 启动飞书机器人（可选） | `start_bot.bat` |
